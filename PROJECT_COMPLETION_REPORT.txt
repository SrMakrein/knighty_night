════════════════════════════════════════════════════════════════════════════════
                         QUICK CUSTOMIZATION GUIDE
════════════════════════════════════════════════════════════════════════════════

All parameters are easily customizable by editing values in the scripts.

MOVEMENT SPEED:
- File: scripts/player/PlayerController.gd
- Find: max_walk_speed = 5.0
- Change to: 8.0 for faster, 3.0 for slower
- Also change: max_run_speed = 8.0

JUMP HEIGHT:
- File: scripts/player/PlayerController.gd
- Find: jump_force = 5.0
- Change to: 7.0 for higher, 3.0 for lower

CAMERA DISTANCE:
- File: scripts/player/CameraController.gd
- Find: distance = 3.0
- Change to: 5.0 for farther, 2.0 for closer

CAMERA HEIGHT:
- File: scripts/player/CameraController.gd
- Find: height = 1.5
- Change to: 2.0 for higher, 1.0 for lower

CAMERA SMOOTHNESS:
- File: scripts/player/CameraController.gd
- Find: smoothing = 5.0
- Change to: 8.0 for smoother, 3.0 for snappier

INPUT SENSITIVITY:
- File: scripts/player/CameraController.gd
- Find: gamepad_sensitivity = 2.0
- Change to: 3.5 for faster, 1.0 for slower

ADD PLATFORMS:
- Open: scenes/levels/main.tscn
- Right-click Main → Add Child Node → StaticBody3D
- Add MeshInstance3D and CollisionShape3D children
- Position and scale as needed

All changes take effect after F5.

                       TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

GAME WON'T START:
- Make sure Godot 4.6 or newer is installed
- Check that all files are intact
- Check console for error messages (View → Output)

CHARACTER DOESN'T MOVE:
- Verify Input Map in Project Settings → Input Map
- Check that PlayerController.gd is attached to Player
- Look for console errors

CAMERA IS JERKY:
- Increase smoothing value in CameraController.gd
- Try values like 7.0 or 8.0

CONTROLLER NOT WORKING:
- Connect Xbox controller to PC
- Check Input Map (Project Settings → Input Map)
- Verify controller is recognized in Windows

ANIMATIONS DON'T PLAY:
- Check animation names in AnimationController.gd
- Verify knight.blend has animations exported
- See DEVELOPER_GUIDE.md for animation setup

More troubleshooting in QUICK_REFERENCE.md!
