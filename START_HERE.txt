Want to change something? It's easy!

Make Character Faster
├─ Open: scripts/player/PlayerController.gd
├─ Find: max_walk_speed = 5.0
├─ Try: 8.0 for faster, 3.0 for slower
└─ Save & Press F5

Adjust Jump Height
├─ Open: scripts/player/PlayerController.gd
├─ Find: jump_force = 5.0
├─ Try: 7.0 for higher, 3.0 for lower
└─ Save & Press F5

Change Camera Distance
├─ Open: scripts/player/CameraController.gd
├─ Find: distance = 3.0
├─ Try: 5.0 for farther, 2.0 for closer
└─ Save & Press F5

Improve Camera Smoothness
├─ Open: scripts/player/CameraController.gd
├─ Find: smoothing = 5.0
├─ Try: 8.0 for smoother, 3.0 for snappier
└─ Save & Press F5

Add More Platforms
├─ Open: scenes/levels/main.tscn
├─ Right-click Main → Add Child Node
├─ Add StaticBody3D with MeshInstance3D and CollisionShape3D
├─ Position in viewport
└─ Save & Press F5
